Path to code: https://github.com/evilChipmunk/ML2

The code should be able to be ran by installing the basic numpy, scikit, and pandas libraries.

Data items are loaded into github. 

In addition, both MlRose and MlRose-hiive are needed to run.

They can be located at:
https://github.com/hiive/mlrose
https://mlrose.readthedocs.io/en/stable/

And installed by using pip install mlrose | mlrose-hiive


The main file, main.py can be ran for all optimization and neural network problems at one go.
Path name in data.py *basePath* will need to be modified to point to the data folder. When running, without setting a base path in each file (onemax, flipflop, fourpeaks, neural2) the default will be to show the plot from matplotlib.
